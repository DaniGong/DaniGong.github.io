Pyramid Dev Environment Setup - Sat, May 28, 2016

  1. Download Setuptool: http://peak.telecommunity.com/dist/ez_setup.py
  2. Install Setuptool: sudo python ez_setup.py
  3. Install virtulenv: sudo easy_install virtualenv
  4. Install Pyramid: virtualenv —no-site-packages env
  5. cd env
  6. bin/easy_install pyramid
  7. Will contain pyramid, template(Chameleon、Mako), PasteDeploy

Deploy Hugo Site on GitHub Pages - Sat, May 28, 2016

  • 1. Create a Repository: danigong.github.io (replace danigong with your own github username)
  • 2. Change directory to root directory of the site.
  • 3. hugo --theme=hyde --baseUrl="http://danigong.github.io/"

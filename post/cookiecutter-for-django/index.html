cookiecutter for django

Mon, Mar 5, 2018

cookiecutter 这是一个专门用于创建项目目录结构的工具，同时也是使用Python来实现的，它可以使用一个模板来创建一个项目，比如上面我介绍的项目目录结构实际上是一个模板，也是github开源的，链接：https://github.com/pydanny/cookiecutter-django

安装cookiecutter pip install cookiecutter

创建项目 cookiecutter https://github.com/pydanny/cookiecutter-django

运行上面第二个命令的时候，会问你一大堆问题，这些问题中下示例（附简短讲解）

Open Sublime with Terminal

A simple way to open sublime with terminal:

  1. Open terminal.
  2. ln -s /Applications/Sublime\Text\2.app/Contents/SharedSupport/bin/subl /usr/local/bin/subl
  3. Use subl command to edit a file.
